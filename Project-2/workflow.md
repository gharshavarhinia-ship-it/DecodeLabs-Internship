# Git Workflow

## Step 1
Create a local repository.

```
git init
```

## Step 2

Create project files.

## Step 3

Check repository status.

```
git status
```

## Step 4

Stage files.

```
git add .
```

## Step 5

Commit changes.

```
git commit -m "Initial Commit"
```

## Step 6

Connect remote repository.

```
git remote add origin <repository-url>
```

## Step 7

Push to GitHub.

```
git push -u origin main
```
