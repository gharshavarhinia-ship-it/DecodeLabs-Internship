# Version Control with Git

## Project Overview
This project demonstrates the basic Git workflow using Git and GitHub.

## Objectives
- Create a Git repository
- Track project changes
- Commit changes
- Push repository to GitHub
- Understand the Git workflow

## Tools Used
- Git
- GitHub
- VS Code
- Windows Command Prompt

## Git Workflow

Working Directory
        ↓
git add
        ↓
Staging Area
        ↓
git commit
        ↓
Local Repository
        ↓
git push
        ↓
GitHub Repository
