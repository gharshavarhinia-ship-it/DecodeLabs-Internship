# Git Cheat Sheet

## Repository

```bash
git init
```

Creates a new Git repository.

---

## Status

```bash
git status
```

Shows changed files.

---

## Add Files

```bash
git add .
```

Stages all files.

---

## Commit

```bash
git commit -m "message"
```

Saves a snapshot.

---

## Push

```bash
git push
```

Uploads commits to GitHub.

---

## Pull

```bash
git pull
```

Downloads latest changes.

---

## Branch

```bash
git branch
```

Lists branches.

---

## Log

```bash
git log
```

Displays commit history.
